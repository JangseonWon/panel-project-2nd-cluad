## 패널검사 워크리스트 및 검사 시리얼 정보 관리 모듈

### 사용 시나리오
- 첫 화면에서 워크리스트 목록을 보여줌
    - 검색을 하면 워크리스트 목록이 조회됨
        - Search 파라미터로 조회
    - 워크리스트는 워크리스트 프로젝트에서 생성한 패널검사의 워크리스트와 1:1 대응됨
    - 워크리스트 이름, 검체수, 생성시각, 생성자 출력
- 이름이나 검체수를 클릭하여 워크리스트 내부 진입
- 시리얼 생성
    - 처음에는 워크리스트 각 검체에 시리얼이 부여되지 않음
    - 시리얼 생성 버튼을 누르면 최근 시리얼을 확인하여 +1 연번 저장
    - 시리얼 그룹에 따라 연번 부여

### 기능
- 워크리스트 검색
- 워크리스트 내 검체 정보 출력
- 최근 시리얼 조회
- 시리얼 저장
